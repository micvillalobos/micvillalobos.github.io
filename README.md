# My Personal Web Profile

A personal webpage made as a mini-project for CMSC 207: Web Programming and Development

---

## What This Project Is

This is a basic one-page website about me. It includes:
- A header with a navigation menu
- An About Me section with a profile picture
- A Hobbies section with hobby descriptions and hobby pictures
- A button that shows random fun facts using JavaScript
- A Contact section with links
- A Footer

---

## Files Included

```
index.html       - the main webpage
style.css        - the stylesheet for colors and layout
script.js        - the JavaScript for the button
images/          - folder for the profile picture and hobby pictures
README.md        - this file
```

---

## Hover Effects (CSS)

- Nav links — text turns gold on hover
- Profile picture — grows slightly bigger on hover using transform: scale(2)
- Hobby boxes — border changes from green to maroon on hover
- Hobby images — grows slightly bigger and border changes to maroon on hover
- Button — background changes maroon to green and grows slightly on hover
- Contact links — color changes from maroon to green on hover
